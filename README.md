# Tomato Leaf Disease Detection with Voice Assistant

## Project Overview

This project is an AI-based tomato leaf disease detection system developed using Deep Learning and Flask. The system helps farmers identify tomato leaf diseases from uploaded leaf images and provides treatment recommendations with voice assistance.

---

## Features

- Tomato leaf disease prediction
- Deep learning hybrid model
- Flask web application
- Grad-CAM heatmap visualization
- Voice assistant using gTTS
- Image upload functionality
- Disease causes and treatment recommendations

---

## Technologies Used

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- NumPy
- HTML/CSS
- SQLite
- gTTS

---

## Disease Classes

The system detects:

- Blight
- Healthy
- Leaf Spot
- Mildew

---

## Workflow

1. User uploads tomato leaf image
2. Image preprocessing is performed
3. Deep learning model predicts disease
4. Confidence score is generated
5. Grad-CAM heatmap highlights affected regions
6. Treatment recommendations are displayed
7. Voice assistant provides audio guidance
